# atmega328p-digital-prayer-clock
Digital prayer time clock built on ATmega328P featuring 7-Segment with HC595 display, DS3231 RTC, EEPROM-based offset storage, button debounce, and FSM-driven user interface.

## Features
- DS3231 Real Time Clock
- Prayer time display LED 7-Segment
- Time and date configuration
- Prayer time offset adjustment
- EEPROM persistent storage
- Button debounce handling
- Finite State Machine (FSM) user interface
- Modular driver architecture

## Hardware
- ATmega328P
- DS3231 RTC
- 7-Segment Display
- HC595 Shift register ic
- Push Buttons

## Development Environment
- Atmel Studio
- SimulIDE
